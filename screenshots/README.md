# Screenshots

此目录用于存放 GitHub 展示截图。

当前已包含：

- `language-selection.png`：多语言选择界面
- `cleanup-modes.png`：清理模式说明界面
- `cleanup-log-example.png`：清理记录示例

## 截图建议

为了保证 GitHub 展示效果，建议：

- 使用较清晰的 100% 或 125% 缩放截图
- 尽量避免暴露用户名、设备名、绝对路径
- 尽量避免展示敏感应用数据
- 如包含扫描结果，优先使用脱敏示例
- 截图内容尽量和当前版本 UI 保持一致

## 维护建议

如果 UI 文案、布局或模式说明发生明显变化，请同步更新截图，避免 GitHub 首页展示与当前版本不一致。
